# Technology Stack

The Heguru Ambassador platform is built with a modern, full-stack JavaScript/TypeScript ecosystem focused on performance, scalability, and mobile responsiveness.

## Core Frameworks

- **[Next.js 15/16](https://nextjs.org/)**: The backbone of the application. We use the App Router for optimal routing, Server Components for performance, and Server Actions for data mutations.
- **[React 19](https://reactjs.org/)**: Powers the interactive user interface, utilizing the latest hooks and features for a smooth experience.

## Backend & Database

- **[Prisma (ORM)](https://www.prisma.io/)**: Used to interact with the database. It provides type-safe queries and handles schema migrations.
- **[PostgreSQL](https://www.postgresql.org/)**: A high-performance, open-source relational database.
- **[Neon (Serverless Postgres)](https://neon.tech/)**: Used for efficient scaling and branching in a serverless environment.

## Styling & UI Components

- **[Tailwind CSS](https://tailwindcss.com/)**: Utility-first CSS framework for rapid UI development and consistent design tokens.
- **[Framer Motion](https://www.framer.com/motion/)**: Integration for high-performance entry and state-transition animations.
- **[Lucide React](https://lucide.dev/)**: For a clean and consistent icon set.
- **[Sonner](https://sonner.stevenly.me/)**: For sleek, non-intrusive toast notifications.

## Mobile Platform

- **[Capacitor](https://capacitorjs.com/)**: Allows us to wrap the web application as a native Android and iOS app, providing access to device features like Push Notifications.

## Other Integrated Tools

- **[Date-fns](https://date-fns.org/)**: For robust date and time manipulation.
- **[Zod](https://zod.dev/)**: For schema-based validation of server actions and API responses.
- **[bcryptjs](https://github.com/dcodeIO/bcrypt.js)**: For secure password hashing.
- **[jsPDF](https://github.com/parallax/jsPDF)**: For client-side generation of certificates and ID cards.
- **[Recharts](https://recharts.org/)**: For dynamic data visualization and analytics charts.
